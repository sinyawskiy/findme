### Demo
Demo [https://sinyawskiy.github.io/findme/](https://sinyawskiy.github.io/findme/)

# Usage
## Preparation
```bash
$ git clone https://github.com/sinyawskiy/findme.git
```

## Package installation
```bash
$ yarn install
```

## Use development server

```bash
$ yarn start
```

## Build assets

```bash
$ yarn build
```
